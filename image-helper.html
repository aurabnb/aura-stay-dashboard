Placeholder Images Setup Instructions

Step 1: Create the directory structure

Make sure the following directory exists:

public/properties/placeholders/

Step 2: Save your images with these exact names:

1. Overwater Bungalows Image →

public/properties/placeholders/overwater-bungalows.jpg

Use the beautiful overwater bungalows image for Aura Properties

2. Safari Lodge Image →

public/properties/placeholders/safari-lodge.jpg

Use the safari lodge with giraffes image for Samsara Properties

3. Vineyard Villa Image →

public/properties/placeholders/vineyard-villa.jpg

Use the vineyard villa image for Airscape Properties

Step 3: Restart your development server

After saving the images, restart your dev server to see the changes.

✨ What you'll see:

  • 🚫 "Coming Soon" badges on Aura, Samsara, and Airscape Properties buttons
  • ⚠️ Buttons are disabled to show they're not available yet
  • 📢 Yellow notification banner when you click on coming soon categories
  • 🖼️ Beautiful placeholder images for each property type
